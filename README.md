## Introduction

This project investigates patterns in radiology service utilization by analyzing differences in patient wait times and procedure volumes between CT scans and MRI scans throughout the year.

The primary research question guiding this analysis is:

**"How do patients' wait times and procedure volumes for CT scans compare to MRI scans across different quarters of the year, and are there identifiable seasonal trends in radiology throughput?"**

The goal of this study is to determine whether certain times of the year experience increased demand for specific imaging modalities. Identifying these seasonal trends can provide insight into workflow efficiency, resource allocation, and potential impacts on patient care within radiology departments.

To answer this question, data will be collected from publicly available healthcare datasets. Using Python, the data will be cleaned, organized, and analyzed with tools such as pandas for data manipulation and visualization libraries like matplotlib or seaborn to illustrate trends and comparisons.

This project demonstrates the application of data analysis techniques, programming skills, and effective communication of findings through both visualizations and documentation.
